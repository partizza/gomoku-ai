### The example of Gomoku game written in python.
Game mode
- vs palyer
- vs AI (chat GPT)

  #
> [!TIP]
> **To start on Linux:**
- add OpenAI token to env variables
```shell
export OPENAI_API_KEY=your-toke-here
```

- run game
```shell
python3 gomoku.py
```
